# Unity Procedural House Generator

A procedural house generation tool for Unity that helps quickly create modular house layouts for game environments.

## Overview

This project is an early version of a Unity tool designed to speed up the creation of houses and building layouts.
The generator can be used for prototyping game environments, level design, and procedural world-building workflows.

## Features

* Procedural house generation inside Unity
* Modular building workflow
* Useful for fast environment prototyping
* Designed for game development projects
* Easy to import using a `.unitypackage` file

## Current Status

This is the first public version of the tool.
The project is still in active development, and more features will be added over time.

## Installation

1. Download the `.unitypackage` file from the `UnityPackage` folder.
2. Open your Unity project.
3. Go to `Assets > Import Package > Custom Package`.
4. Select the downloaded package.
5. Import all required files.

## Usage

After importing the package into Unity, add the generator component to a GameObject in your scene and configure the available parameters in the Inspector.

More detailed documentation will be added in future versions.
